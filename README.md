# 🎡 Color Game

An interactive full-stack web application where users spin a wheel to receive a random color.

## 🚀 Features
- 🎯 Random color generation via backend API
- 🔍 Searchable zone selection (dropdown + input)
- 🔐 Admin panel for managing players and colors
- 🌐 Deployed online

---

## 🧠 Recent Improvement

Replaced manual zone input with a searchable dropdown to improve user experience.

---

## 🐛 Bug Encountered

The app broke due to a mismatch between frontend and backend variables:

- Frontend used: `zone`
- Backend expected: `zones`

---

## ✅ Fix

Aligned variable naming across the application to restore API communication.

---

## 💡 Lessons Learned

- Consistency is critical in full-stack development  
- Small bugs can have large impacts  
- Debugging requires tracing data flow step-by-step
